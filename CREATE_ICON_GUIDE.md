# ساخت آیکون MediaStar Remote Control

## 🎨 طراحی آیکون

### مشخصات فنی
- **فرمت**: ICO (Windows Icon)
- **اندازه‌ها**: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256
- **رنگ**: 32-bit (با alpha transparency)
- **نام فایل**: `icon.ico` در پوشه اصلی پروژه

### ایده‌های طراحی

#### 1. آیکون ساده و مدرن
```
📺 + 📡 = 🎯
یک تلویزیون با امواج رادیویی و هوش مصنوعی
```

#### 2. طراحی حرفه‌ای
- **پس‌زمینه**: آبی تیره/مشکی (تطابق با UI)
- **عنصر اصلی**: آنتن ماهواره یا تلویزیون
- **جزئیات**: امواج WiFi/Bluetooth، نماد AI (🤖)
- **رنگ‌ها**: آبی، سفید، نقره‌ای

#### 3. طراحی مینیمال
- **شکل**: مستطیل با گوشه‌های گرد
- **نماد**: حرف M (MediaStar) با آنتن
- **رنگ**: گرادینت آبی

## 🛠️ روش‌های ساخت

### روش 1: استفاده از ابزار آنلاین (توصیه شده)
1. به سایت [favicon.io](https://favicon.io) بروید
2. تصویر یا متن خود را آپلود کنید
3. آیکون را با اندازه‌های مختلف دانلود کنید
4. فایل `icon.ico` را در پوشه اصلی پروژه کپی کنید

### روش 2: استفاده از GIMP (رایگان)
1. GIMP را نصب کنید
2. فایل جدید با اندازه 256x256 بسازید
3. طراحی خود را انجام دهید
4. File → Export As → `icon.ico`
5. در پوشه اصلی پروژه ذخیره کنید

### روش 3: استفاده از Adobe Illustrator
1. طراحی وکتور بسازید
2. File → Export → Export As
3. فرمت ICO را انتخاب کنید
4. تمام اندازه‌ها را انتخاب کنید

### روش 4: استفاده از IcoFX (پرداخت)
1. نرم‌افزار IcoFX را نصب کنید
2. طراحی با ابزارهای حرفه‌ای
3. خروجی با تمام اندازه‌ها
